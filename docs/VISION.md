# VISION.md — 项目愿景

## 产品定位

个人基金投资管理系统，基于 E大（ETF拯救世界）投资理念，帮助管理基金持仓、网格策略、交易记录和投资分析。

## 目标用户

投资者本人（单用户模式）。

## 核心价值

1. **持仓透明** — 从交易记录派生批次持仓，实时查看浮动/已实现盈亏
2. **网格策略** — 自动化网格交易执行、状态追踪、盈亏计算
3. **投资分析** — 资产配置可视化、技术指标、回测
4. **数据主权** — 所有数据存储在自己的 Supabase 实例，支持导出/备份

## 可交付标准（对齐共识）

各维度评分 ≥7/10，构建+测试全绿，核心流程无明显痛点。

## 当前阶段

v2.7.0，核心功能已实现，进入质量打磨阶段。
